## Overview

A browser-based slot machine game built purely with vanilla HTML, CSS, and JavaScript. Spin the reels, match symbols, and win — no libraries, no frameworks.

## Tech Stack

- **HTML** — structure
- **CSS** — styling and reel animations
- **JavaScript** — game logic, DOM manipulation, and event handling

## What I Learned

This was a fun side project that helped me get comfortable with vanilla JavaScript — specifically DOM querying, event listeners, randomness, and managing simple game state without any framework.
