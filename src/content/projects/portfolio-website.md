## Overview

This is my personal portfolio website, the very site you're reading now. It was built to showcase my projects and serve as a central hub for my work as a frontend developer.

## Tech Stack

- **Astro** — static site generator with component islands
- **Tailwind CSS** — utility-first styling

## What I Learned

Building this site gave me hands-on experience with Astro's content collections, component-driven architecture, and deploying a static site to GitHub Pages.
