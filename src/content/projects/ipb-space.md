## Overview

IPB Space is an end-to-end facility booking platform built as the final project for my System Design course. It allows users to search, reserve, and manage campus facility bookings, with a backend designed around clean architecture principles.

## My Role

I served as **Backend & DevOps Engineer**, responsible for the full server-side architecture, infrastructure, and deployment pipeline.

## What I Did

- Architected the backend using **FastAPI**, adhering to clean code principles and a modular layered architecture.
- Designed custom booking logic with queue handling to resolve database concurrency conflicts.
- Containerized all services with **Docker** and published interactive API contracts via **Insomnia** for cross-team collaboration.
- Integrated **JWT authentication** and automated email notifications via **SMTP (Mailtrap)**.
- Managed file storage with **Appwrite** and deployed the full stack on **Railway**.

## Tech Stack

- **FastAPI** — Python async web framework
- **Docker** — containerization and service orchestration
- **JWT** — authentication
- **Appwrite** — object storage
- **Mailtrap (SMTP)** — automated notifications
- **Railway** — cloud deployment
