## Overview

Orlin Farm is a business website built for a local farm as the final project for my Software Engineering (RPL) course. The site covers product listings, ordering, and basic content management for the farm owner.

## My Role

I served as a **backend developer**, implementing the server-side logic, database schema, and API endpoints using Laravel.

## Tech Stack

- **Laravel** — PHP web framework
- **MySQL** — relational database
- **Blade** — Laravel templating engine

## What I Learned

This project deepened my understanding of MVC architecture, database relationships, and working within a structured software engineering process (requirements, design, implementation, testing).
