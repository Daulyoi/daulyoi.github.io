## Overview

Andamata is a commercial web profile built for a travel agency client via CodePanda. The goal was a fast, SEO-friendly site that the client could maintain themselves through a CMS, with a design translated directly from Figma blueprints.

## My Role

Solo Frontend Developer — responsible for the full build from blank canvas to production delivery.

## What I Did

- Built the web profile from scratch using **Astro.js** and **SCSS** for a high-performance static output.
- Utilized **Static Paths** and SEO optimization to improve build times and search engine rendering efficiency.
- Translated high-fidelity **Figma** blueprints into semantic, accessible code layouts.
- Implemented dynamic rich text rendering via **CMS** integration, allowing the client to update content without touching code.

## Tech Stack

- **Astro.js** — static site generator with component islands
- **SCSS** — modular, maintainable styling
- **Figma** — design handoff
- **CMS** — client-managed content
