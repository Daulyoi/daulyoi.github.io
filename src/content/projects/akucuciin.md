## Overview

AkuCuciin is an e-commerce logistics startup providing a laundry pickup-and-delivery service. I contributed to the web platform as a **Frontend Developer**, building the core order-placement experience for a high-traffic production application.

## My Role

Frontend Developer, responsible for the order flow UI and real-time map integration.

## What I Did

- Engineered a responsive order-placement flow using a **Bento Grid** layout to organise complex UI clearly on all screen sizes.
- Implemented real-time location mapping with **Leaflet.js** so customers can pin pickup and delivery addresses.
- Conducted API testing using **Postman** to validate integration with backend services.
- Maintained production code that supported the business transition into a self-sustaining, actively operating campus enterprise.

## Tech Stack

- **React.js** — component-based UI
- **Leaflet.js** — interactive real-time maps
- **Bento Grid** — responsive layout system
- **Postman** — API testing
